# IT-Helpdesk-Lab-Series
IT-Helpdesk-Lab-Series is a virtualized home-lab setup using VirtualBox that simulates a small enterprise IT environment.   It demonstrates Active Directory management, domain-joining PCs, remote administration, permissions &amp; group policy, software deployment, helpdesk workflows, and common troubleshooting scenarios.
